# [sos](../../)
#### console API abstraction for the browser or server

## API ([0.0](../../releases))

```js
var sos = require('sos')
```

### sos()
#### `sos(method, ...arguments)`

- `sos.dir(object)`
- `sos.log(message)`
- `sos.info(message)`
- `sos.warn(message)`
- `sos.error(message)`
- `sos.assert(expression, message)`
- `sos.trace()`
- `sos.clear()`
